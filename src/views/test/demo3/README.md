| 特性         | Transferable Objects         | SharedArrayBuffer             |
| ------------ | ---------------------------- | ----------------------------- |
| **所有權**   | 轉移（主執行緒失去存取權）   | 共享（雙方同時擁有）          |
| **記憶體**   | 同一塊記憶體，但只能一方存取 | 同一塊記憶體，雙方同時存取    |
| **同步需求** | 無需同步（獨佔存取）         | 需要 Atomics 同步（競態風險） |
| **使用後**   | 發送方無法再使用             | 雙方都可持續讀寫              |
| **安全要求** | 無特殊要求                   | 需要 COOP/COEP 響應頭         |
| **典型場景** | 大數據一次性轉移             | 持續協作、頻繁更新            |
